# 智慧校园管理系统 前台任意文件上传

FOFA:

```
body="DC_Login/QYSignUp"
```

在用户注册页面(无注册按钮可直接访问接口)上，上传功能导致可以上传恶意文件，直接getshell.

路径：

```
/DC_Login/QYSignUp
/DC_OA_WJG/Upload
```
